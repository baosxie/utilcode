退货说明

  1. 1. mmall7天无理由退货服务规则:
    1. （1）自签收之日起7日内，mmall为您提供无理由退货服务，支持不同包裹单独退货。
    2. （2）商品如无质量问题：请保证商品及商品本身包装为mmall出售时原装且配件赠品资料齐全，不影响二次销售。
    3. （3）商品如存在质量问题，请自签收之日起7日内提出退货申请，过期将不予办理。
    4. （4）如您购买时索要发票且发票已寄送给您，请将发票随商品一同寄回。
    5. （5）为节省您的退货运费和保证退款时效，一张订单仅能办理一次退货，暂不支持二次退货。
  2. 2. 以下情况不予办理退货
    1. （1）订单已超过退货规则期限
    2. （2）任何非mmall出售的商品（序列号不符）；
    3. （3）已经超过三包保修期的商品；
    4. （4）因非正常使用及保管导致出现质量问题的商品
    5. （5）非质量问题，但原包装已被损毁或配件赠品不全的商品
    6. （6）无法提供商品的发票（如已索要发票）、保修卡等三包凭证或者三包凭证信息与商品不符及被涂改的；
    7. （7）礼包或套装中的商品不可以部分退换货；
    8. （8）其他依法不应办理退换货的（详见各类商品退货细则）。
  3. 3. 温馨提示
    1. （1）退货回寄货物时，请不要选择平邮或者到付方式，否则将不办理退货
    2. （2）因商品质量问题产生的退货，退货运费由mmall承担；因您个人原因产生的退货，退货运费由您个人承担。为提升客户体验，mmall为您订单中每个原包裹的退货提供10元运费补贴，红包有效期为30天。
    3. （3）退货退款金额按照商品的实际结算金额退款，优惠券将按退货商品实付进行退还
    4. （4）非整单退货，已使用优惠券的订单，优惠券将不予退还。
    5. （5）如需了解详细退货规则及流程，请点击查看帮助中心售后服务内容。
