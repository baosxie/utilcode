一、退换货流程

（1）请拨打对应供应商电话，联系退换货 —— 供货商上门退换货

（2）请拨打客服电话，联系退换货—— 供货商上门退换货

二、退换货服务细则：

1. 第三方商户收到商品的剩余保质期没有高于商品整体保质期三分之二以上，进口商品剩余保质期没有高于整体保质期二分之一以上，收货方可以拒收或者在收货后要求供应商退、换货。

2. 当商品外包装破损影响销售时，收货方可以要求供应商退换或拒收。如收货时为整件，拆零销售时发现商品外包装破损，收货方可以要求供应商换货或退货。

3. 鼠损不能退换货。

4. 促销商品在活动期间可以换货，不可以无理由退货。特殊促销商品退换货服务细则，促销期间会特别单独说明。

5. 假冒伪劣商品，收货方可以要求供应商立即退货并第一时间通知百万店客户经理或者致电0571-87651759 进行投诉。

6. 保质期内其他质量问题（如真空包装食品漏气、发霉、变质、有异物等等），收货方可以要求供应商退货或者换货。如供应商不予退换，收货方可以第一时间通知百万店客户经理或者致电0571-87651759 ，客服将在收到投诉信息后24小时内进行处理。
