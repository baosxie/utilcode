购物指南：

1.交易条款：

买家在接受商品订购与送货的同时，有义务遵守以下交易条款。您在百万店下订单之前或接受百万店送货之前，请您仔细阅读以下条款：

1.1 订购的商品价格以您下订单时百万店网上价格为准。

1.2 请清楚准确地填写您的真实姓名、送货地址及联系方式。因如下情况造成订单延迟或无法配送等，百万店及卖家将不承担责任：

A. 客户提供错误信息和不详细的地址； 

B. 货物送达无人签收，由此造成的重复配送所产生的费用及相关的后果； 

C. 不可抗力，例如：自然灾害、交通戒严、突发战争等。 

1.3安全性：在订购商品的同时，我们会保证交易信息的安全。 

1.4隐私权：百万店尊重您的隐私权，在任何情况下，我们都不会将您的个人和订单信息出售或泄露给任何第三方（国家司法机关调取除外）。我们从网站上或电话中得到的所

有客户信息仅用来处理您的相关订单。 

1.5免责：如因不可抗力或其它百万店无法控制的原因使百万店销售系统崩溃或无法正常使用导致网上交易无法完成或丢失有关的信息、记录等，百万店会尽可能合理地协助处

理善后事宜，并努力使客户免受经济损失。 

1.6客户监督：百万店希望通过不懈努力，为客户提供最佳服务，百万店在给客户提供服务的全过程中接受客户的监督。 

关于订单：

1.订单查询：

1.1有哪些订单状态：

一个百万店的新订单从下单到订单完成，会经历各种状态，我们会将各种状态显示在订单详情页面，希望以此种方式让您更好的了解订单情况，及时跟踪订单状态，打消疑虑并顺利完成订货。以下是订单状态的简单说明：

a.待发货：指您提交订单后，但卖家尚未发货的订单；

b.待收货：指卖家已安排配送，但买家尚未确认收货的订单；

c.已完成：指买家确认收货并付款完成的订单。

2.订单修改/取消：

您在百万店提交的订单，暂不支持线上修改或者取消，如需修改或者取消，请致电百万店客服电话0571-87651759

3.订单确认：

请您收货后点击确认收货，如果无法点击，可能是系统故障，请您稍后再试。

配送与签收：

1.发货时间：

1.1卖家应当在百万店规定的时间内发货，但有特殊规定或与买家协商好的除外。

2.发货信息的确认：

2.1除买卖双方另有约定外，卖家应当按照订单约定的收货地址、收货人信息发货，并负责将商品送达到买家指定的收货地址。

2.2如商品需要买家到指定地点提取的，需在发货前告知买家并征得买家同意。卖家违反前述规定的，买家有权拒绝签收商品。

2.3卖家在发货后应及时在百万店平台更新发货信息。

3.收货信息的填写与变更：

3.1买家应当在订单中向卖家提供准确的收货地址和收货人信息。买家在提供收货人信息时，可以选择本人或者他人作为收货人。买家选择他人作为收货人，该收货人违反本规范约定义务的，由买家承担相应责任。

3.2买家需要变更订单中的收货地址或收货人信息的，应当在商品发出前与卖家取得联系征得卖家的明确同意，或协商进行处理。

3.3因买家填写的收货地址和(或)收货人信息不准确，或者未经卖家同意要求变更收货地址或收货人信息，导致卖家发货后商品无法送达的，运费由买家承担。

4.商品配送：

4.1卖家所选择的物流在运送商品时应当主动联系收货人，而不应在未征得收货人同意的情况下将商品交由小区保安、门卫、公司前台等人签收。

5.商品签收：

5.1卖家按照约定发货后，收货人有及时收货的义务。收货人可以本人签收商品或委托他人代为签收商品，被委托人的签收视为收货人本人签收。

5.2买家只填写了收货地址，但没有填写收货人或填写的收货人信息不特定，商品在收货地址被签收的，该签收视为消费者本人签收。

6.商品验收：

6.1收货人签收商品时，应当对商品进行验收。

6.2收货人签收商品时发现数量或商品不一致的，有权拒绝签收商品。 卖家应在商品拒收后及时联系买家，进行处理。

6.3收货人无正当理由拒绝签收商品的，运费由买家承担。

6.4商品毁损、灭失等的风险自收货人签收商品后由卖家转移给收货人。
