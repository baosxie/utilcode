入驻指南

一、供应商入驻需要提供的资质文件

（1）需提交公司营业执照副本、组织机构代码证副本、税务登记证副本（如是一般纳税人资格的，请提供相应资格证明）、银行开户许可证（复印件均需加盖企业公章/电子文件需保证图片清晰）、法人的身份证复印件，食品流通许可证（经营食品的供应商提供）。

（2）上传产品资质：提交公司所经营品牌对应的资质文件，包括品牌资质、品牌授权及品牌对应的产品资质（复印件均需加盖企业公章/电子文件需保证图片清晰）。

二、如何与采购商进行商品沟通

百万店提供在线客服及供应商联系方式，方便与采购商进行商品沟通。

三、百万店是否自行提供物流配送服务？

百万店在发展初期，不考虑自建仓库和配送体系。货物的配送由供应商自行解决或由百万店推荐的物流服务商进行配送。

四、百万店收到货款后什么时候打钱给供应商？

（1）百万店提供现金结算和网银及第三方支付平台结算

（2）使用现金结算，百万店不参与相关结算过程，付款过程中的纠纷与责任由供货商和采购商自行协商解决。

（3）使用网银及第三方支付平台结算，供应商按订单送货并经收货方书面签收后，可向百万店申请提取货款，提取货款一般1~3各工作日到账。
