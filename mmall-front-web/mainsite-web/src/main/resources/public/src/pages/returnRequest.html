退货说明

  1. 1. 自您签收商品之日起7天内，XX为您提供无理由退货服务。规则如下
    1. （1）产品质量无问题的：如果商品无质量问题，您要求退/换货，请您保证商品外包装完整（包含商品外包装盒上的透明封装胶纸）
    2. （2）产品质量有问题的：如果您在使用过程中对商品质量表示置疑并要求退/换货，需要出具有效的书面鉴定
    3. （3）如您购买时索要发票，一经退货请将发票随商品一同寄回，如发票丢失将无法办理退货
  2. 2. 以下情况不予办理退货
    1. （1）收货期已超过30天的产品
    2. （2）非质量问题，但包装盒已被开封的商品（包含撕开商品外包装盒上的透明封装胶纸）
    3. （3）非由MISS出售的商品
    4. （4）因非正常使用及保管导致出现质量问题的商品
    5. （5）免费赠品及试用装礼品
    6. （6）促销时专门标注的不可退换的商品（如商品保质期在6个月之内的清仓商品）
    7. （7）礼包或套装中的商品不可以部分退/换货
  3. 3. 温馨提示
    1. （1）MISS保证出货时货物外包装的完好无缺。为了保证您的利益，请您在收到货物时务必验货，请拆开包装箱，确认货品完好元损后再签收。如果发现货物在运输途中有破损， 您有权拒收，在快递单上写明拒收原因，让配送人员退回货物。在签收后配送人员离开现场，再发现货物损坏，我们概不负责。
    2. （2）您在退换货品的途中，有责任保证所退还货品的安全及完整，如因退还途中出现丢包、损坏（影响二次销售的），责任由顾客承担。
    3. （3）MISS收到发生退换货的产品包裹后，我们会对退回的商品进行质检，确认商品及外包装无损后，再根据需求进行退换货操作
