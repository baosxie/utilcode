如何使用优惠券？

在购物车点击“确认结算”，进入“确认订单信息”页面后，请仔细确认订单信息后，在页面结算金额处找到“使用优惠券”，选择“我的可用优惠券”或“手动输入优惠券代码”，即可获得相应优惠。 同时需要注意，每次活动的内容不同，优惠券的使用期限也不同，请及时抵扣使用。通过优惠券购买的产品享受同样的退换货条款。
